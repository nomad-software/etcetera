nothrow @nogc size_t hash(T)(T value, size_t seed = 0) if (isNumeric!T || isBoolean!T || isSomeString!T || is(T == struct) || is(T == union) || is(T == class));

Return a hash of any value.

Parameters

T value The value to hash.
size_t seed An optional seed for the hash.

Return Value

A numeric hash.

Examples

assert(hash(1337) == 3412859167);
assert(hash("foo") == 4138058784);
