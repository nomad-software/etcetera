pure nothrow @nogc @safe T unquote(T)(T text) if (isSomeString!T);

Remove single, double or back quotes from around a passed string. Strings will only be unquoted if they start and end with matching quotes.

Parameters

T text The string to remove quotes from.

Return Value

If the above rules apply the unquoted string, otherwise it's returned unchanged.

Examples

assert(unquote(`"Lorem ipsum dolor sit amet."`) == "Lorem ipsum dolor sit amet.");
assert(unquote("'Lorem ipsum dolor sit amet.'") == "Lorem ipsum dolor sit amet.");
assert(unquote("`Lorem ipsum dolor sit amet.`") == "Lorem ipsum dolor sit amet.");
